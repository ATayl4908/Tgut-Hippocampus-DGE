# ZB-DGE-analysis

R script, associated data, and figures for T. guttata hippocampal differential expression project. 

- 'rsem_GENE.txt' contains mapped transcript counts (output by RSEM, delivered by UW Biotechnology Center).
- 'SampleTable.txt' contains informations about the categorization of samples in this analysis.
- 'ZB-Social Isolation data analysis script.R' contains data analysis script. See inline documentation for more details.
